## 使用建议

淡斑产品最好在晚上用,因为有些美白成分见光容易失效。洁面后,先用精华,再用面霜,最后记得涂防晒。白天防晒做不好,晚上再怎么淡斑都白搭。

视黄醇产品刚开始用可能会有点脱皮,这是正常的建立耐受过程。建议从隔天用一次开始,等皮肤适应了再每天用。孕期和哺乳期的女性要避免使用视黄醇产品。

眼膜可以放冰箱冷藏,敷的时候凉凉的,消肿效果更好。熬夜第二天早上敷一片,能让眼睛看起来精神很多。
